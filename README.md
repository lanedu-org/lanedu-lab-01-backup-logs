# LAB-01 · El jefe necesita el archivo YA

## Historia
Tu lead te escribió por chat:
> “Necesito un backup limpio de logs antes de las 18:00, el auditor está aquí”.

## Objetivo
Crear un script que comprima los logs de `/var/log`
(o una carpeta simulada) y genere un archivo `.tar.gz`
con la fecha actual en el nombre (YYYY-MM-DD).

## Reglas
- No borres archivos originales.
- Incluye comandos o snippets que usaste.
- Adjunta evidencia de ejecución.

## Entrega
Crea un Pull Request contra este repositorio con tu solución.

⚠️ No pegues links en la plataforma.
La entrega se valida automáticamente desde GitHub.
